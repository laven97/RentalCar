# 🚗 RentCar — Сервіс Оренди Автомобілів

Сучасний, швидкий та зручний вебдодаток для оренди автомобілів. Проєкт створений для того, щоб користувач міг без зайвих зусиль підібрати ідеальне авто за різними критеріями та миттєво забронювати його на потрібну дату.

---

## ✨ Особливості проєкту (Features)

* 🔎 **Розумний пошук:** Зручна фільтрація автомобілів за брендом, погодинною ціною та пробігом (із кастомними випадаючими списками).
* 📅 **Кастомний календар:** Інтерактивний віджет вибору дати, розроблений з нуля, який автоматично підлаштовується під дні тижня та закривається при кліку поза його областю.
* 📝 **Автозбереження чернетки:** Якщо користувач випадково перейде на іншу сторінку під час заповнення форми, введені дані не втратяться завдяки глобальному менеджменту стану. При уході з сайту форма автоматично очищається.
* 🔔 **Миттєві сповіщення:** Плавні Toast-нотифікації, які з'являються у кутку екрана лише після успішної відповіді сервера про бронювання.
* ⚡ **Оптимізація запитів:** Усі дані кешуються та оновлюються у реальному часі без зайвих перезавантажень сторінок.

---

## 🛠️ Технологічний стек (Tech Stack)

Додаток побудований на базі найсучасніших інструментів фронтенд-розробки:

* **Framework:** Next.js (App Router, Client/Server Components)
* **Library:** React 
* **State Management:** Zustand (для збереження чернеток форми та стану авто)
* **Data Fetching & Mutations:** TanStack React Query (для асинхронних запитів до API)
* **Styling:** CSS Modules (ізольовані стилі для кожного компонента)
